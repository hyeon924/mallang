# 말랑이 프로젝트

## 📌 개요  
**말랑이**는 유튜브 API를 기반으로 한 언어학습플랫폼 프로젝트입니다.

---

## 🛠️ 기술 스택  
- **Next.js**  
- **Tailwind CSS**  
- **TypeScript**

---
